# magic-pigeon 🐦

A learning project about building a local AI agent, kept in two versions so the evolution is visible.

| Version | Path | What it is |
|---------|------|------------|
| v1 | `Version1/` | The original single-file Python agent, terminal only. |
| v2 | `Version2/` | The same agent refactored into a reusable engine, with a web UI, an OpenAI-compatible gateway, web search, and a model picker. |

The current version lives in `Version2/`. To run it, start there:

```bash
cd Version2
```

and follow `Version2/README.md`. I built this mainly to understand how coding agents work under the hood, not as a polished replacement for tools like Claude Code.
